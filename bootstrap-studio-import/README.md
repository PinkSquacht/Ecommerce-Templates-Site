# Bootstrap Studio Import Bundle

This folder contains a ready-to-import static snapshot based on your current storefront design direction.

## Files
- `index.html`: the page to import into Bootstrap Studio.
- `assets/studio-import.css`: custom style layer used by the page.

## Import Steps
1. Open Bootstrap Studio.
2. Create a new design.
3. Use the HTML import option and select `index.html` from this folder.
4. If prompted, keep relative asset paths.
5. Verify `assets/studio-import.css` is included in the design.

## Notes
- This is a static design snapshot for visual editing.
- React routing, context, and API logic are not included in this import package.
